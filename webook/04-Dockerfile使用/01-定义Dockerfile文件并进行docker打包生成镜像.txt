1. 将应用打包好对应系统的执行文件，当前打包出的文件为 webook 文件（linux可执行二进制文件）
go env -w GOOS=linux
go env -w GOARCH=arm
go build -o webook .

2. 编写 Dockerfile 文件，docker build 命令会基于该文件定义的指令进行执行和打包生成镜像

# 基础镜像
FROM ubuntu:20.04

# 将编译后的linux文件 webook 复制(打包进来)到镜像，放到工作目录 /app 目录下
COPY webook /app/webook
# 设定我的工作目录
WORKDIR /app

# CMD 是执行命令，可以镜像启动之前执行一些命令
# ENTRYPOINT 进来就执行这条命令
ENTRYPOINT ["/app/webook"]


2. 执行 docker build 命令，根据Dockerfile打包生成对应的镜像文件
